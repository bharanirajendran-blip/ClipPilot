# ClipPilot Lite - Backend API

Production-ready FastAPI backend for short vertical video generation using AI.

## Architecture

### Tech Stack
- **FastAPI** - Modern Python web framework
- **Supabase** - PostgreSQL + Auth + Storage
- **Redis + ARQ** - Async job queue for video processing
- **External APIs**:
  - Anthropic Claude (Research, Script, Review, Metadata agents)
  - Runway Gen-3 Alpha Turbo (Video generation)
  - ElevenLabs Turbo v2.5 (Text-to-speech)
  - Deepgram Nova-2 (Speech-to-text for captions)
  - FFmpeg (Video assembly and processing)

### Pipeline
1. **Input Validation** - Guardrail check (12+ compliance)
2. **Rate Limiting** - Per-user lifetime + global daily limits
3. **Content Research** - Claude researches topic
4. **Script Generation** - Creates timed video script
5. **Quality Review** - Iterative review loop (up to 2 revisions)
6. **Shot List** - Visual prompts optimized for Runway
7. **Video Generation** - Runway creates scenes
8. **Narration** - ElevenLabs generates speech
9. **Captions** - Deepgram transcribes to SRT
10. **Assembly** - FFmpeg combines video/audio/captions/disclaimer
11. **Delivery** - Supabase Storage hosting

### Safety
- 12+ audience guardrails at input and output
- Blocked content: violence, explicit, hate speech, misinformation
- Auto-disclaimer: "Generated by ClipPilot based on user input. Verify claims independently."
- Family-friendly default configurations

### Job Management

**Create Job**
```bash
POST /api/jobs
Content-Type: application/json
Authorization: Bearer {token}

{
  "topic": "How solar panels work",
  "style": "educational",
  "duration": 30
}

Response:
{
  "job_id": "uuid",
  "status": "queued",
  "message": "Video generation queued"
}
```

**Get Job Status**
```bash
GET /api/jobs/{job_id}
Authorization: Bearer {token}

Response:
{
  "job_id": "uuid",
  "status": "processing",
  "progress": 45,
  "current_step": "Generating video",
  "created_at": "2024-04-21T10:00:00Z",
  "updated_at": "2024-04-21T10:02:30Z",
  "error_message": null
}
```

**Get Job Result**

## Error Handling

### Common Errors

**429 Too Many Requests**
- User has exceeded lifetime limit (2 videos)
- Global daily limit reached (20 videos)

**400 Bad Request**
- Topic failed safety check
- Job not ready for requested operation

**404 Not Found**
- Job doesn't exist
- User not authorized to access job

**500 Internal Server Error**
- Video generation failed
- Service integration error
- Database connection issue

## Troubleshooting

### Worker not processing jobs
- Check Redis connection: `redis-cli ping`
- Verify worker is running: `python worker.py`
- Check ARQ queue: `python -c "import arq; print(await arq.get_worker(redis_url)...)"`

### Supabase connection errors
- Verify SUPABASE_URL and keys are correct
- Check network connectivity to Supabase
- Verify RLS policies are correctly configured

### Video generation timeouts
- Increase JOB_TIMEOUT_SECONDS
- Check Runway API status
- Verify FFmpeg is installed and in PATH

### Low quality videos
- Adjust Claude prompts for better visual descriptions
- Check Runway model selection
- Verify video resolution settings in FFmpeg
